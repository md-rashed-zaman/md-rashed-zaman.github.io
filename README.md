# md-rashed-zaman.github.io

Personal portfolio for **Md. Rasheduzzaman**.

- Next.js (App Router) + Tailwind CSS
- shadcn/ui-style component system
- Static export (`output: "export"`) for GitHub Pages

## Getting Started

Install deps and run the dev server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build (static export)

```bash
npm run build
```

The static output is generated into `out/`.

## Content

Portfolio content lives in `app/data.ts`.
